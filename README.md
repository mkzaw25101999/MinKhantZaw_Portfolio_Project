# MinKhantZaw_Portfolio_Project

# Portfolio Website

A personal portfolio website showcasing my software engineering projects and skills.
Built with HTML and CSS.

## About

This is my personal portfolio website created to showcase my web development and data analysis projects. The site features a modern dark theme with ocean-blue and cyan gradients.

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Animated sections with fade-in effects
- Project showcase with tech stack tags
- Contact information with social links

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling with gradients, flexbox, and grid layout
- **GitHub Pages** - Deployment and hosting

# Sections

- **Hero** - Introduction and call-to-action buttons
- **About** - Background, education, and objectives
- **Contact** - Email and social media links
- **Skills** - Technical skills and competencies
- **Projects** - Portfolio of completed projects with descriptions

# Setup & Installation

If you want to run this locally:

1. Clone the repository:
   git clone https://github.com/mkzaw25101999/MinKhantZaw_Portfolio_Project

2. Navigate to the project folder:

3. Open `index.html` in your browser

- No build process or dependencies required!
